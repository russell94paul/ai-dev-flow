# AI Dev Flow System

## Overview

ai-dev-flow is a structured AI-driven development workflow system.

It transforms software development from ad-hoc prompting into a repeatable process.

---

## Core Workflows

### new-project
Breaks a project idea into structured features.

### feature
Runs a full development lifecycle:

1. Grill (clarify requirements)
2. PRD generation
3. Implementation planning
4. TDD execution

---

## Philosophy

- Structured thinking over ad-hoc prompting
- Human-in-the-loop decision making
- Persistent knowledge (Obsidian-style artifacts)
- Iterative improvement of workflows

---

## Architecture

- CLI orchestrates workflows
- Skills define behavior (prompt modules)
- Claude executes workflows
- Obsidian-style files store outputs

---

## Future Direction

- Self-improving workflows
- Prompt evolution
- Multi-agent collaboration
- Persistent developer memory system
